## 周期性任务

### C001: 确定性机会交易

**类型**: 周期性 (每次检查时执行)
**状态**: 执行中
**创建时间**: 2025-12-21

**规则**:

1. **最大仓位限制**: 同时最多持有 3 个币种 (含挂单中的开仓单)
2. **杠杆限制**: 整体账户杠杆不超过 10X
3. **选币策略**: 结合 BOS 突破交易法, 筛选确定性高的交易机会
   - 优先选择结构清晰、信号明确的币种
   - 需有明确的入场点、止损位、止盈目标
   - 综合考虑成交量、持仓量、资金流向等因素
4. **执行方式**: 每次检查行情时, 筛选符合条件的币种, 评估后执行交易

**挂单管理**:

- 下挂单后必须更新 [当前挂单记录](#当前挂单记录) 区域
- 每次检查时需核对: 挂单是否仍在? 条件是否仍有效? 是否过期?
- 挂单成交或取消后, 从记录中移除

**开仓挂单有效期**:

- **开仓限价单**: 默认有效期 1 小时 (API 自动设置 GTD)
- **平仓挂单** (止损/止盈): 无需设置有效期, 使用 GTC 长期有效
- 币安会自动取消过期的开仓挂单, 无需手动处理
- 如需更长有效期, 可在下单时手动指定 `goodTillDate` 参数

**终止条件**: 交易员手动取消
