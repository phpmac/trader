# 交易时段配置

## 时区

**中国标准时间 (CST / UTC+8)**

## 可交易时段

| 时段 | 开始 | 结束 | 说明 |
|-----|------|------|------|
| 日间交易 | 07:00 | 24:00 | 可开仓、可平仓 |
| 夜间休息 | 00:00 | 07:00 | 禁止开仓, 仅可平仓 |

## 开仓规则

1. **只在可交易时段开仓** (07:00 - 24:00 CST)
2. **夜间 (00:00 - 07:00) 禁止开新仓位**
3. **任何时间都可以平仓** (止损/止盈不受时段限制)

## 收盘时间

- **日收盘**: 24:00 CST
- 收盘前应检查持仓状态, 根据"过夜持仓规则"决定是否平仓

## AI 代理行为

### 开仓前检查

```
当前时间 (CST) 在 07:00 - 24:00 之间?
├── 是 → 可以开仓
└── 否 → 禁止开仓, 即使有开仓信号也不执行
```

### 收盘前检查 (23:30 - 24:00)

如果有持仓, 检查 `docs/最新指令.md` 中的过夜持仓规则:
- 有"过夜不持仓"禁令 → 检查盈亏, 赚钱则平仓
- 无相关禁令 → 可持仓过夜

## 时间获取

使用 MCP 提供的时间工具获取当前时间:

- **MCP 服务**: `time-tools`
- **工具**: `get_current_time` - 获取当前 CST 时间

## 注意事项

1. 此配置是**基础约束**, 优先级高于任务
2. 夜间有强烈信号也不开仓 - 纪律第一
3. 如需调整时段, 交易员明确说明后修改此文件
