# 交易教训

## 概述

此 skill 存放历史交易失误案例, **开仓前必须检查**, 避免重蹈覆辙.

## 使用方法

**开仓前**, 必须:

1. 读取 `cases/` 目录下的所有案例
2. 对照当前交易计划, 检查是否存在类似错误倾向
3. 如发现类似情况, **必须警告交易员**

## 常见错误类型速查

### 1. 单一指标依赖

**特征**: 只看一个指标就开仓 (如 CVD、RSI、MACD)

**正确做法**: 多维度确认 (结构 + 资金流 + 情绪)

### 2. 追涨杀跌

**特征**: 看到涨就追多, 看到跌就追空

**正确做法**: 等待回调至支撑/阻力位入场

### 3. 关键位置前开仓

**特征**: 在关键阻力/支撑位前就急着开仓

**正确做法**: 等待 BOS/CHoCH 突破确认后, 在回调时入场

### 4. 情绪化交易

**特征**: 被短期波动影响, 缺乏耐心

**正确做法**: 严格按计划交易, 不满足条件不开仓

## 开仓前检查清单

在准备开仓时, 必须逐项确认:

- [ ] 是否只依赖单一指标? (必须多维度确认)
- [ ] 是否在追涨杀跌? (应等待回调)
- [ ] 是否在关键位置前开仓? (应等待突破确认)
- [ ] 是否情绪化决策? (应按计划执行)
- [ ] 是否设置了止损止盈? (必须设置)
- [ ] 风险收益比是否 > 1.5:1? (必须满足)

**任何一项不满足, 禁止开仓!**
