# 交易方法整理

本目录用于整理和存放各种交易方法、交易思路和交易策略.

## 目录结构

每个交易方法都是一个独立的子目录, 包含:
- `README.md` - 交易方法的详细说明
- 可选的辅助文件 (图表示例、回测数据等)

## 当前收录的交易方法

| 方法名称 | 描述 | 路径 |
|----------|------|------|
| BOS突破交易法 | 基于 LuxAlgo Strong/Weak HL 的 BOS 突破交易策略 | `BOS突破交易法/` |

## 使用说明

1. 每个交易方法都是独立的, 可以单独学习和应用
2. 交易方法应配合"固定风险仓位管理"技能使用
3. 所有方法都需要先在模拟环境验证后再实盘使用
4. 交易思路仅供参考, 实际交易需根据市场情况灵活调整
