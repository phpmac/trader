# BOS突破交易法

基于 [LuxAlgo](https://docs.luxalgo.com/docs/getting-started/introduction) 付费脚本的 BOS (Break of Structure) 突破交易策略.

## 交易周期

- **分析周期**: 15 分钟 (15m) - 用于信号识别和交易决策
- **交易类型**: 日内交易 (Intraday) - 当日开仓当日平仓
- **持仓时间**: 通常数小时, 不隔夜

## 策略分类

- **突破交易** - 基于结构突破信号入场
- **趋势交易** - 顺趋势方向交易, 盈亏比好时可持仓跟随趋势 (日内)

## 核心工具

- **LuxAlgo Price Action Concepts** - 付费脚本
- **Strong/Weak HL** - 阶段性顶底 (用于固定风险仓位管理计算最大止损)
- **BOS 信号** - 结构突破信号
- **HH/LL** - 提前止损点和止盈目标

## 核心概念区分

| 概念 | 作用 |
|------|------|
| **Strong/Weak High** | 阶段顶部, 固定风险仓位管理的最大止损参考 |
| **Strong/Weak Low** | 阶段底部, 固定风险仓位管理的最大止损参考 |
| **HH (Higher High)** | 提前止损点 (做空) / 止盈目标 (做多) |
| **LL (Lower Low)** | 止盈目标 (做空) / 提前止损点 (做多) |

---

## 做空交易

### 信号条件

1. 价格在 Strong/Weak High (顶部) 区域形成 **HH (Higher High)**
2. 随后出现 **跌破 BOS 信号** - 结构被打破
3. 此时可准备做空

### 入场方式

- **BOS 线** 作为参考开仓价
- 通常使用 **限价单** 在 BOS 线附近挂单等待成交
- **不建议** 直接市价进场

### 止损设置

- **最大止损**: Strong/Weak High (阶段顶部) - 用于固定风险仓位管理计算仓位
- **实际止损**: HH 最高点 (提前止损, 更优的风险控制)

### 止盈策略

根据盈亏比 (R:R) 决定止盈方式 (使用 HH 止损计算盈亏比):

#### 情况一: 盈亏比 < 1.5

- 使用 **LL (Lower Low)** 作为止盈目标
- 设置市价止盈, **一次性全部平仓**

#### 情况二: 盈亏比 >= 1.5

- 可设置 **分批止盈** (2次+)
- 第一批: LL 位置止盈部分仓位
- 第二批+: 以下选项根据行情选择:
  - 移动止损至成本价, 让利润奔跑
  - 等待更优价格重新开仓
  - 拿趋势交易, 追踪更大行情

## 关键要点

1. **必须配合固定风险仓位管理** - 使用 Strong/Weak H/L 计算最大风险
2. **HH/LL 用于实际执行** - 提前止损获得更好的风险控制, 作为止盈目标
3. **限价单优于市价单** - 获得更好的入场价格
4. **盈亏比决定止盈策略** - 低 R:R 一次性平仓, 高 R:R 分批止盈
5. **灵活应对市场** - 具体操作需根据当时行情判断
6. **动态调整止盈止损** - 持仓期间根据新出现的 HH/LL/BOS 信号调整

## 持仓期间的动态调整

**每次检查行情时**, 必须评估是否需要调整止盈止损:

### 止损调整

| 场景 | 操作 |
|------|------|
| 出现新的 HH (做空时) | 评估是否需要调整止损至新 HH |
| 出现新的 LL (做多时) | 评估是否需要调整止损至新 LL |
| 达到 1R 盈利 | 考虑移动止损至成本价 |
| 出现反向 BOS 信号 | 考虑立即平仓或收紧止损 |

### 止盈调整

| 场景 | 操作 |
|------|------|
| 趋势延续, 出现新的 LL/HH 目标 | 考虑延后止盈, 追踪更大行情 |
| 趋势减弱, 未达目标但接近 | 考虑提前止盈, 落袋为安 |
| 出现反向信号但未触发止损 | 考虑提前止盈, 不等目标 |
| 接近强支撑/阻力区域 | 评估是否分批止盈 |

### 调整原则

1. **止损只收紧不放宽** - 保护本金, 不扩大风险敞口
2. **止盈根据行情灵活调整** - 强势行情可延后, 弱势行情宜提前
3. **每次调整需通知** - 发送 Slack 通知说明调整原因

## 注意事项

- 此策略需要 LuxAlgo 付费订阅
- 建议先在模拟环境验证交易逻辑
- 不同市场/品种可能需要参数调整
- 结合整体市场结构和趋势分析效果更佳
- **日内交易**: 避免隔夜持仓, 收盘前清仓
- **15分钟周期**: 所有信号分析基于 15m K线
